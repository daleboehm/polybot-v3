## Bugs Found & Fixed

### CRITICAL

| # | Bug | Impact | Fix |
|---|-----|--------|-----|
| 1 | **Gamma API path-based URLs** — 8 occurrences across 7 files using `/markets/{id}` instead of `?conditionId={id}` | All 19 position evaluations returning 422 errors. Exit engine completely blind — could not evaluate stop losses, profit targets, or resolutions. | Replaced all 8 path URLs with query param format + added array unwrapping (API returns `[{...}]` not `{...}`) |
| 2 | **Gamma API array response not unwrapped** — 12 total occurrences (8 original + 4 found in QA) | `resp.json()` returned a list but code called `.get()` expecting a dict. Silent failures everywhere. | Added `_resp_data[0] if isinstance(_resp_data, list)` unwrap pattern in all 12 locations |
| 3 | **USDC contract address wrong in capital_router.py** | capital_router had `0x...cc7DA6FF30ef` instead of the correct `0x...9449Aa84174`. Could cause failed USDC balance queries or routing to wrong contract. | Corrected to match all other files |

### HIGH

| # | Bug | Impact | Fix |
|---|-----|--------|-----|
| 4 | **16 market_maker.py zombie processes** | Each cron run spawned a new instance that never exited. Accumulating indefinitely, consuming memory. | Killed all zombies + added `timeout 120` to cron entry |
| 5 | **master-dashboard.service crash loop** | Rogue manual aggregator.py holding port 9090, preventing systemd service from binding. | Killed rogue process, restarted systemd service |
| 6 | **auto_ops.py false CRITICAL alerts** | Used `exec()` to validate auto_trader.py, which fails on `__file__` references. Reported import failure every 5 minutes. | Switched to `py_compile` for safe syntax checking |

### MEDIUM

| # | Bug | Impact | Fix |
|---|-----|--------|-----|
| 7 | **Corrupt JSON state files** (resolution_alerts.json, edge_signals.json) | Cron `>` redirect not atomic — concurrent runs produced invalid JSON with mixed content. Dashboard reads would fail. | Fixed corrupt files + converted cron to atomic write pattern (write to /tmp, then `mv`) |
